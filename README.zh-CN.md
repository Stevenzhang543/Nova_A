## 构建要求

  要构建并运行此项目，请确保已安装以下环境：
* **Node.js**：`v18.0.0` 或更高（现代 Vite/Vue 3 环境所需）
* **npm**：`v9.0.0` 或更高（或者使用 `yarn v1.22+` 或 `pnpm v8+`）
* **Vue**：`^3.3.0`（通过 package.json 自动解析）
* **TypeScript**：`~5.0.0` 或更高（通过 package.json 自动解析）
* **Cargo**：`1.71.0` 或更高。Cargo edition 2021。
* **WASM-pack**：`1.71.0` 或更高（随 cargo 自动安装）

## 构建指南

### 1. 初始化（安装依赖）
  在首次运行引擎之前，你需要安装所有所需的 node 模块。进入 `Nova_A` 根目录并运行：

* **npm install pnpm**
* **pnpm install**
* **pnpm install tauri**

### 2. 初始化（构建后端引擎）
  进入 Nova_A\nova_core 目录并运行：

* **wasm-pack build --target web**

### 3. 开发构建（本地服务器）
  启动一个带有热模块替换（HMR）的本地开发服务器，以便快速迭代测试：

* **pnpm tauri dev**

（这通常会在 http://localhost:1420 启动编辑器）。

### 4. 生产发布（构建）
  为生产部署编译、打包并压缩应用：

* **pnpm tauri build**

（这将生成一个 dist/ 文件夹，其中包含静态 HTML、CSS 和优化后的 JS 资源）。


## 物理属性

### TRANSFORM（变换）

  **Position（位置）**
    物体在屏幕上的位置。就像把一个玩具放在桌子上——你决定它左右（x）和上下（y）的位置。范围：无限*。
  **Rotation（旋转）**
    物体旋转的角度。就像在桌面上旋转一本书。0° 表示正常，90° 表示侧放。范围：-180 到 180 度。

### MOTION（运动）

  **Linear Velocity（线速度）**
    物体移动的速度和方向。就像扔一个球——它会向前移动，可能还会向上。范围：无限*。
  **Angular Velocity（角速度）**
    物体旋转的速度。就像旋转的硬币。正值表示一个方向，负值表示相反方向。范围：无限*。
  **Linear Damping（线性阻尼）**
    随时间减慢运动，就像空气阻力。值越高，物体停止得越快。范围：0 到 1。
  **Angular Damping（角阻尼）**
    随时间减慢旋转，就像轮子的摩擦。范围：0 到 1。

### MASS & PHYSICAL BEHAVIOR（质量与物理行为）

  **Mass（质量）**
    物体的重量。越重的物体越难推动。就像羽毛和石头的对比。范围：0.1 到 100。
  **Moment of Inertia（转动惯量）**
    物体旋转的难易程度。长棍比小球更难旋转。自动计算。
  **Gravity Scale（重力系数）**
    控制重力对物体的影响程度。1 表示正常重力，0 表示漂浮，2 表示下落更快。范围：0 到 5。

### FORCES（力）

  **Force（力）**
    施加在物体上的推力。就像推动一个箱子——更大的力会让它移动得更快。范围：无限*（x 和 y）。
  **Torque（扭矩）**
    使物体旋转的扭转力。就像在门的边缘推门。范围：无限*。

### SHAPE & SIZE（形状与大小）

  **Size / Radius（尺寸 / 半径）**
    物体的大小。更大的物体更容易发生碰撞并占据更多空间。范围：无限*。

### MATERIAL（材质）

  **Restitution（弹性）**
    物体的弹性程度。0 表示没有弹性（像粘土），1 表示非常有弹性（像橡胶球）。范围：0 到 1。
  **Static Friction（静摩擦）**
    接触其他表面时开始移动的难易程度。就像推动一个不容易动的重箱子。范围：0 到 1。
  **Dynamic Friction（动摩擦）**
    物体在已经移动时抵抗滑动的程度。就像在冰面和砂纸上滑动的区别。范围：0 到 1。

### COLLISION STATE（碰撞状态）

  **Is Static（是否静态）**
    如果开启，物体将完全无法移动。就像墙或地面。
  **Is Kinematic（是否运动学）**
    物体仅在你控制时移动，而不是由物理驱动。就像游戏中的移动平台。

## RENDERING（渲染）

  **Color (RGB)（颜色）**
    使用红、绿、蓝值表示物体的颜色。就像混合颜料。每个值控制该颜色的强度。范围：每个通道（R, G, B）为 0 到 255。
  **Transparency (Alpha)（透明度）**
    物体的透明程度。0 表示完全不可见，1 表示完全不透明。范围：0 到 100。

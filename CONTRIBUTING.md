Thank you for your interest in contributing!

### 1. Fork the repository
Create a copy under your GitHub account.

### 2. Create a feature branch
git checkout -b feature/my-new-feature

### 3. Commit changes
Follow commit message rules:
- `feat:` new feature  
- `fix:` bug fix  
- `refactor:` no-behavior changes  
- `docs:` documentation  

### 4. Submit a Pull Request
Ensure your PR:
- Builds successfully  
- Passes code formatting  
- Includes documentation updates if needed
